# FIFA World Cup 2026 — Signal Room

A real-time, interactive intelligence dashboard for the FIFA World Cup 2026.

## Features

- **Today's Matches**: Live match cards with team flags, GMT times, venues, and AI predictions
- **Group Standings**: All 12 groups with live tables, form indicators, and qualification status
- **Tournament Stats**: Goals, scorers, and interactive Chart.js visualizations
- **Match Deep Dive**: Click any match for detailed analysis — predictions, head-to-head, player stats, cross-references, and probability charts
- **Team Deep Dive**: Click any team in standings for full stats, upcoming matches, results, and top scorers
- **Favorite Team**: Select your team from the dropdown for instant access
- **Data Sources**: Attribution for all open data sources used

## Data Sources

- [worldcup26.ir](https://worldcup26.ir) — Free open-source REST API
- PredictLix — Match predictions and odds
- RotoWire — Expert betting analysis
- openfootball — Public domain schedules
- StatsBomb — Free event-level data with xG
- football-data.org — Free fixtures and results API

## Tech

- Pure HTML/CSS/JS (single file, no build step)
- [Chart.js](https://www.chartjs.org/) for visualizations
- Google Fonts (Inter + Oswald)
- Responsive design (mobile-first)
